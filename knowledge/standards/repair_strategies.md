 # 分型面尖钢修复策略速查

 ## 一、策略总览

 | 策略名称 | 适用场景 | NX API | 复杂度 | 风险 |
 |---------|---------|--------|--------|------|
 | add_material_patch | 孤立的薄钢/尖角区域 | UF_MODL_create_block1 + unite | 低 | 可能改变外观 |
 | offset_face | 大面积薄钢面 | UF_MODL_create_face_offset | 中 | 可能引起面干涉 |
 | blend_edges | 锐边尖角 | UF_MODL_create_edge_blend | 低 | 安全 |
 | move_parting_line | 分型线附近尖钢 | 需重建分型面 | 高 | 影响模具结构 |
 | local_thicken | 局部壁厚不足 | 偏置+布尔 | 中 | 需注意过渡 |

 ## 二、策略选择规则

 ```text
 诊断结果
   │
   ├─ 策略标记为 "add_material_patch"
   │   └─ 在尖钢区域中心创建补块 → 布尔求和到母体
   │
   ├─ 策略标记为 "offset_face"
   │   └─ 对标记的面执行外偏置 0.5~1.5mm
   │
   ├─ 策略标记为 "blend_edges"
   │   └─ 对该区域锐边倒 R0.3~R1.0 圆角
   │
   └─ 策略标记为 "review_only"
       └─ 只记录，不修复
 ```

 ## 三、参数参考

 | 严重程度 | 偏移增厚量 | 倒圆角半径 | 补块大小 |
 |---------|-----------|-----------|---------|
 | CRITICAL | 1.2mm | R0.8 | bbox 外扩 50% + 1mm |
 | WARNING | 0.8mm | R0.5 | bbox 外扩 30% + 0.5mm |
 | INFO | 0.3mm | R0.2 | 仅记录，不修复 |

 ## 四、多轮修复策略调整

 | 轮次 | 策略 |
 |------|------|
 | 第1轮 | 检测 → 修复所有 CRITICAL + WARNING |
 | 第2轮 | 重新检测 → 修复剩余的 CRITICAL，自动加大偏移量 20% |
 | 第3轮 | 重新检测 → 对仍不过的 CRITICAL 切换策略 |
 | 第4轮+ | 报告人工，标记为不可自动修复 |

 ## 五、注意事项

 - 每次修复后必须重新运行检测，不允许跳过
 - 同一个区域在 3 轮后仍然 FAIL，应切换到不同的修复策略
 - 修复操作应在原 PRT 副本上执行，保留原始备份
 - 偏移量不宜超过 2.0mm，过大会影响装配关系

## 六、fix_centroid 精确定位机制（v2 新增）

从 v2 版本开始，检测 DLL 输出两个质心：

| 字段 | 含义 | 用途 |
|------|------|------|
| centroid | 合并后区域的 bbox 中心 | 统计报告 |
| fix_centroid | 区域内最严重那张面的真实质心 | 修复 DLL 找面用 |

fix DLL 优先使用 fix_centroid 定位目标面，找不到时回退到 centroid。
这解决了"区域 bbox 中心不在任何面上"导致的修复落空问题。
